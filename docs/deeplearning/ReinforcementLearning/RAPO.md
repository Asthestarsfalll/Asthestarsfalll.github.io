**RIRM机制**：有三个reward
- Rans：是否首次就找到了正确答案
- Rver：反思步骤数量是否合理
- Racc：最终答案

**RAPO**：
- 多维价值函数：
	- 反思感知奖励
	- 轨迹熵，token熵平均
	- 新颖度，提取最后一层嵌入，与历史轨迹嵌入计算余弦相似度
	- 冗余度，同上
	- 参数自适应，前期低奖励高熵的权重提高，中期一致，后期高奖励低熵优先
- 动态过滤
	- 剔除奖励低的无效样本
	- 对奖励相近的样本做k-means压缩
	- 剔除奖励异常、轨迹突变的样本
- 优先级排序
	- top-k采样
	- 优先采样token熵前20所在轨迹
- 梯度只更新 token 熵前20% 的
