去重庆听惘闻xMONO的演出，并且在有限的预算内顺带玩/吃（以吃为主）三天（19-22）。

## 第一天(下午)

到了估计下午5点左右了，景点啥的都关了，可能随便逛逛，晚上去八一美食街吃点东西。



## 第二天

依然在渝中半岛，路线：
```
解放碑->湖广会馆->白象居->八一好吃街->国泰艺术中心->吃火锅->洪崖洞
```
解放碑 800米可以到罗汉寺（门票10, 可是五点就关门了），再走1.1公里到达湖广会馆（学生票12.5, 同样结束咧），再走800到白象居，可以同时拍到来福士、 长江索道、市井居民楼。再走1公里到达八一好吃街（可以用于第二天解决午餐），再走600米到达国泰艺术中心（拍个照得了），上人行天台走500米到魁星楼，下来走400米吃老火锅。吃完去洪崖洞，往上坡的方向到达戴家巷步道，可以通畅进入观景台，观景台拍照得了，洪崖洞内部应该全是人吧。还有经历可以逛一下解放碑附近的商场咯。

## 第三天

21号咧，切记晚上8点要去蜚声LIVEHOUSE听演出啊。
主要以来于轻轨出行，微信搜索小程序乘车码，不需要购票，扫码即可。
路线：

```
李子坝->梁山鸡->鹅岭二厂->喝喝茶，吃小吃->皇冠大扶梯->江北大剧院->江滩公园->重庆18梯
```
由于江滩公园晚上好看，所以可能直接去18梯吃饭，然后去听演出。

## 第四天（上午）

