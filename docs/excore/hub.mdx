hub内的代码修改于[此]( https://github.com/MegEngine/MegEngine/blob/master/imperative/python/megengine/hub/)，主要作用是可以从 github 等网站上克隆代码到本地cache文件夹并导入指定模块.适用于需要外部模块但是其依赖嵌套较深，或不想在自己的代码中增加额外代码的情况.
:::
