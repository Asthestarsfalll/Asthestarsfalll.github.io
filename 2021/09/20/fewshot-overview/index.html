A Survey on Few-Shot Learning

小样本学习综述

论文名称：Generalizing from a Few Examples: A Survey on Few-Shot Learning

作者：YAQING WANG，QUANMING YAO，JAMES T. KWOK，LIONEL M. NI

摘要

  • 机器学习在数据密集型任务中取得了成功，但是当数据集较小时其往往会受到阻碍；

  • 提出了小样本学习（Few-Shot Learning），利用先验知识，FSL可以快速推广到仅拥有少量监督信息的样本的新任务；本文给出了FSL与几个相关机器学习任务的区别，并且指出FSl的核心问题是经验风险优化器（empirical risk minimizer）不可靠；基于如何使用先验知识来处理这个核心问题，从三个角度对FSL方法进行分类：

    1. 数据，其使用先验知识来增强监督经验；
    2. 模型，其使用先验知识来减少假设空间的大小；
    3. 算法，其使用先验知识来改变对给定假设空间中最佳假设的搜索。

    基于此，讨论每个类别的优缺点，为未来的研究提供见解。

关键词：先验知识，元学习，学习范式

论文有详细介绍

概述

问题定义

首先回顾一下文献当中是如何定义机器学习的：

Machine Learning：A computer program is said to learn from experience E with respect to some classes of task T and performance measure P if its performance can improve with E on T measured by P.

如果一个计算机程序的性能可以通过P来测量E对T的改善，那么就说它可以从经验E中学习某些有关类别任务T和性能度量P

例如一个图像分类任务T，机器学习可以通过对大量标记图像（例如Imagenet数据集）的训练获得的E（Experience）来提高其分类精度P

如此，定义了FSL：

Few-Shot Learning：FSL is a type of machine learning problems (specified by E, T and P), where E contains only a limited number of examples with supervised information for the target T

少样本学习是一种机器学习问题（由E、T和p指定），其中E仅包含有限数量的事例，并且目标T具有监督信息

一些FSL的例子

FSL方法将E中可用的监督信息与先验信息相结合，使得任务T变得可行 。一个非常经典FSL算法就是贝叶斯学习。

当仅有一个样本具有监督信息时，其被称为One-Shot Learning；当没有任何样本具有监督信息时，其被称为Zero-Shot Learning。

相关学习问题

这一节介绍机器学习中一些与FSL相关的问题，以及他们之间的联系和区别：

弱监督学习

弱监督学习从仅仅包含弱监督（比如监督信息不完整、不准确）的E中学习；与FSL最相关的是只有少量样本具有监督信息的弱监督学习。根据使用的是oracle（专家）或是人工干预，可以进一步分为以下几类：

  1. Semi-supervised learning：半监督学习从E中少量的标记样本和大量（通常）的非标记样本当中学习。通常应用于文本或网页分类。
  2. Active learning：主动学习

差异：

  1. 弱监督学习仅包含分类和回归任务，而FSL还包括强化学习；
  2. 弱监督学习主要使用未标注的样本作为E的附加数据，而FSL利用各种先验知识，譬如预训练模型、来自其他领域的监督数据，并且对使用标记数据与否没有限制；

因此，当先验知识是为标记数据、任务是分类或者回归时，FSL就成为弱监督学习问题。

不平衡学习

不平衡学习从E中学习y的偏态分布，解决不平衡问题的一种方式就是通过改变采样方式。

相比之下，FSL通过几个样本对y进行训练和测试，同时可能将其他y作为学习的先验知识。

迁移学习

迁移学习将知识从训练数据丰富的源域/任务转移到训练数据稀缺的目标域/任务，迁移学习在FSL中被广泛应用，先验知识从源任务迁移到FSL任务当中

元学习

元学习通过提供的数据集和元学习元知识提高了任务T的P。

核心问题

机器学习的任何任务都会存在预测误差，本节将会说明有监督学习中基于误差分解的FSL的核心问题。

经验风险最小化

给定一个假设h，我们希望最小化其预期风险R，即相对于P(x,y)测量的损失
